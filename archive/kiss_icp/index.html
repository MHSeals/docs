kiss_icp

kiss_icp is an odometry pipeline 'that just works'. :3

installation

  1. run pip install kiss-icp
  2. that's it! you're done!

running kiss_icp

run ros2 launch kiss_icp odometry.launch.py topic:=/wamv/sensors/lidars/lidar_wamv_sensor/points
